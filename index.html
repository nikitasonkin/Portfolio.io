Projects

Cyber News Bot

CyberNewsBot is a modular data pipeline that ingests RSS feeds, uses newspaper3k to fetch and clean full article text, applies advanced AI-driven NLP with the facebook/bart-large-cnn model for concise summaries on GPU or CPU, performs SHA-256–based deduplication, and ensures concurrency-safe execution and reliable delivery with retry logic to Telegram and Microsoft Teams.

  • View on GitHub

CyberBot Analyst Project

This repository delivers an end-to-end data analysis pipeline for CyberNewsBot. It includes: SQL schema to define normalized tables (PostedNews, SkippedNews, Topics, Articles, Trends) for clean, deduplicated storage. Jupyter Notebooks that demonstrate: JSON ingestion, pandas-driven cleaning and feature engineering, SQLAlchemy loading, BART-based summaries, SentenceTransformer embeddings, and DBSCAN clustering to uncover topic patterns. A final step that writes enriched data back to SQL, ready for interactive Power BI dashboards.

  • View on GitHub

CyberBot PowerBI

This Power BI dashboard builds on the CyberBot Analyst Project, which provided the core SQL schema, Python ETL functions, and Jupyter notebooks for ingesting, cleaning, and analyzing cybersecurity news data.

  • View on GitHub

WWL Sales Analytics

This repository provides a ready-to-use analytics toolkit of 10 advanced T-SQL queries against Microsoft’s Wide World Importers sample database. Delivers insights on yearly revenue, customer performance, product profitability, churn detection, and more—enabling data-driven decision-making.

  • View on GitHub

AdventureWorks Sales Schema

This project showcases the end-to-end creation of a relational database using SQL. It includes a structured schema, data loading from AdventureWorks2022, and analytical queries that highlight the design’s capabilities. Ideal as both a portfolio demonstration and a practical sandbox for database modeling, ETL workflows, and SQL-driven analytics.

  • View on GitHub

Happiness and Health Analysis

This project offers a full analytics pipeline for global happiness and health indicators. It uses Python (pandas, NumPy) for data collection and cleaning, applies statistical methods (correlation, regression) to probe links between social support, corruption, life expectancy, GDP, and happiness, and delivers interactive visualizations with Matplotlib, seaborn, and Plotly.

About

I am a technology entrepreneur driven by a passion for solving real-world problems through code, artificial intelligence, and advanced data analytics. I believe that data science and AI are not passing trends, but the foundation for the next wave of business innovation. My expertise lies in building automated systems—from bots that process and distribute information (such as sending data automatically to Telegram and Teams) to developing advanced data pipelines for in-depth analysis of dynamic information.

Throughout my journey, I have combined knowledge from information systems, cloud infrastructure (AWS, Spark), data engineering (SQL, Python), and business analytics with Power BI—all while mastering machine learning solutions and independently learning new technologies.

I see myself as a pioneer at the intersection of AI, innovation, and business data analysis, motivated by a desire to deliver impactful solutions that drive people and organizations forward.

My goal for the coming years is to continue developing autonomous agents and machine learning modules, deepening my understanding of intelligent systems that can discover new insights, predict trends, and make critical information accessible for decision-making.
